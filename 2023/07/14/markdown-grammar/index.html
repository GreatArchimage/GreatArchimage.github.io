# Markdown 基本语法

Markdown 是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与 HTML 混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown 被大量使用，如 Github、Wikipedia、简书等。

# 标题

在单词或短语前面使用 # 创建标题， # 的数量代表了标题的级别。

# 段落

使用空白行分隔段落。

# 换行

在一行的末尾添加两个或多个空格，然后按回车键，即可创建一个换行 ( <br> )。

# 强调

# 粗体

在单词或短语的前后各添加两个星号或下划线加粗文本。

# 斜体

在单词或短语的前后各添加一个星号或下划线将文本以斜体显示。

# 粗体和斜体

在单词或短语的前后各添加三个星号或下划线加粗文本。

# 代码

使用反引号包裹可将单词或短语表示为 代码 。

要创建代码块，请将代码块的每一行缩进至少四个空格或一个制表符。

#include <stdio.h>
int main()
{
printf("Hello, World!");
return 0;
}

# 分割线

要创建分隔线，请在单独一行上使用三个或多个星号 ( *** )、破折号 ( --- ) 或下划线 ( ___ ) ，并且不能包含其他内容。

# 链接

链接文本放在中括号内，链接地址放在后面的括号中，链接 title 可选。
