# 使用 Github + PicGo 搭建图床

参考教程：使用 Github+picGo 搭建图床，保姆级教程来了 - 知乎 (zhihu.com)

# 创建仓库

  1. 在 Github 创建一个仓库，用于存放图片资源

  2. 在 Github 设置中生成 Token

  3. 下载 PicGo，Releases · Molunerfinn/PicGo (github.com)

  4. 在 PicGo 设置图床

# 测试

在 PicGo 上传图片，上传成功后 Github 仓库应有上传的图片，复制图片链接到浏览器应能访问图片。
